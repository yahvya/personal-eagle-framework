\hypertarget{a01125_autotoc_md4}{}\doxysection{Structure}\label{a01125_autotoc_md4}
\begin{quote}
Le dossier src contient les éléments servant à la création de l\textquotesingle{}application \end{quote}

\begin{DoxyItemize}
\item Le dossier {\bfseries{configs}} contient les fichiers de configurations (environnement, framework, configuration blade / twig ainsi que les fonctions globales à l\textquotesingle{}application)
\item Le dossier {\bfseries{controllers}} vise à accueillir les controllers, par défaut la class abstraite Custom\+Controller y est ajoutée afin de servir de classe mère customisable aux futurs controllers ainsi que le controller d\textquotesingle{}accès à la maintenance par défaut
\item Le dossier {\bfseries{models}} vise à accueillir les models, par défaut la class abstraite Custom\+Model y est ajoutée afin de servir de classe mère customisable aux futurs models
\item Le dossier {\bfseries{public}} visant à accueillir toutes les ressources publiques
\item Le dossier {\bfseries{storage}} représente le dossier de stockage de l\textquotesingle{}application, par défaut contient le fichier hash d\textquotesingle{}accès à la maintenance
\item Le dossier {\bfseries{routes}} contenant les différentes routes de l\textquotesingle{}application
\item Le dossier {\bfseries{treatment}} visant à contenir les class de traitement contient par défaut Custom\+Treatment classe parent customisable des class de traitement
\item Le dossier {\bfseries{views}} racine de recherche des éléments de vue, peut contenir du css et js accessible ~\newline
 
\end{DoxyItemize}
